# 九州 · 殇州试炼

基于 `Vite + TypeScript + Phaser 3` 的九州题材轻量战斗 MVP。当前版本聚焦单章原型：`殇州·北陆荒原`。

## MVP 范围

- 北陆蛮族玩家阵营
- 竖屏单局自动战斗
- 同名单位三合一升阶
- 波次通关后的三选一奖励
- 部族图腾作为第一层九州题材机制

## 本地运行

```bash
npm install
npm run dev
```

## 验证命令

```bash
npm test
npm run build
```
